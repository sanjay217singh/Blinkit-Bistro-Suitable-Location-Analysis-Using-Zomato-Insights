# Blinkit-Bistro-Suitable-Location-Analysis-Using-Zomato-Insights

##  Overview

This project presents an **end-to-end data analytics pipeline** built using the Zomato dataset to identify **optimal restaurant locations** and analyze market opportunities.

It combines **Python-based data analysis** with an **interactive Power BI dashboard** to generate insights on:

* Demand patterns
* Feasibility of new restaurant locations
* Cuisine trends across cities

---

##  Objective

The goal of this project is to:

* Analyze restaurant data across major Indian cities
* Identify **high-demand and high-success localities**
* Provide **data-driven recommendations** for opening new bistro outlets

---

##  Tech Stack

* **Python** (Data Cleaning & Feature Engineering)
* **Jupyter Notebook** (`Zomato_Bistro.ipynb`)
* **Power BI** (`Blinkit_Bistro.pbix`)
* **Dataset**: Zomato Restaurant Data

---

##  Key Features

###  Overview & Key Insights

* Average cost for two
* Average rating
* Demand score
* Feasibility score
* Total number of restaurants

###  City & Locality Intelligence

* City-wise success score comparison
* Locality-level demand analysis
* Geographic visualization of restaurant distribution

###  Cuisine Analysis

* Popular cuisines based on demand
* Average ratings across cuisines
* Cost distribution by cuisine
* Voting trends

###  Recommended Store Locations

* Identifies top-performing localities
* Provides cuisine suggestions for each location
* Uses **success score** to rank expansion opportunities

---

##  Workflow

1. Data collection from Zomato dataset
2. Data cleaning and preprocessing in Python
3. Feature engineering (Demand Score, Success Score, Feasibility Score)
4. Data transformation for visualization
5. Dashboard creation in Power BI

---

##  Key Insights

* Certain localities show **high demand concentration**, making them strong candidates for expansion
* Cuisine preferences vary significantly across cities
* High demand does not always imply high feasibility
* Data-driven location selection improves restaurant success probability
